### 개요

objective-c로 개발된 게임 엔진으로 추후 C++로 언어를 바꾸면서 cocos2d-x가 되었습니다. 기본적으로 Windows, Mac OS, Linux, Android, iOS 등을 지원하고 있으며, 시간이 지나면서 Lua, JavaScript와 같이 다양한 언어로 개발이 가능하게 되었습니다.
